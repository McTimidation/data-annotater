#Project summary

A Django app for data annotation. Requirements are in requirements.txt (Django 5.2.10 listed).

##Prerequisites

Python 3.11+ (recommended)
git
virtualenv or other Python virtual environment manager

###Quick start â€” clone, install, run

Clone the repo git clone https://github.com/McTimidation/data-annotater.git cd data-annotater

Create and activate a virtual environment

macOS / Linux python3 -m venv .venv source .venv/bin/activate
Windows (PowerShell) py -3 -m venv .venv ..venv\Scripts\Activate.ps1
Windows (cmd) py -3 -m venv .venv ..venv\Scripts\activate.bat
Install Python dependencies pip install -r requirements.txt
